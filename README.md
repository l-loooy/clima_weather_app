# clima_weather_app

App includes:

 - Dark mode and vector assets 
 - UITextField 
 - Implement Delegate Design Pattern 
 - Work with openweather's API, making HTTP request with URLSession 
 - Parse JSON with JSONDecoder 
 - CoreLocationApp
